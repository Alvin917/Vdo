# 微豆 Vdo

一个使用 Vue.js 重构 [豆瓣](http://www.douban.com) 的项目  
项目网站 http://ralfz.com/douban/


## 快速使用  

``` bash
# 克隆项目到本地
git clone https://github.com/RalfZhang/Vdo.git

# 安装依赖
npm install

# 在 localhost:8080 启动项目
npm run dev
```

## 教程  


``` bash
npm install -g vue-cli
vue init webpack-simple hello-vue
cd hello-vue
npm install
npm run dev
```
